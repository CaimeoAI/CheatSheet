Basic structure of HTML5 document

Tag Explanation (hover over)

  • <html> </html> This element follows the doctype information, which is used to inform the brwoser that this is an HTML document. You can use the 'lang' attribut with the en value to specify that the document is in English. But nowadays, even the lang attribute is unnecessary for the document to validate or function correctly.
  • <head> </head> The next part is this section. This element contains metadata (document title, character set, styles, links, scripts), specific information about the web page that is not displayed to the user. The <head> element is used to specify the metadata to provide browsers and search engines with technical information about the web page. For example, if you want to specify the author of your document, you may use the <head>.
  • <body> </body> The <body> of a document contains the content of the document. The content may be presented by a user agent in different way. E.g., the content can be text, images links, colors, graphics, etc. Between the body tagsm there can be different elements, to which you can give style by using CSS properties. Just add and id or class selector to your HTML element and in the the <style> section mention your preferred styling options (color, size, font, etc.)
  • <header> </header> Use the <header> element to define a header for the document or section. It usually contains a logo, search bar, navigation links, etc.
  • <nav> </nav> The <nav> element defines a block of navigation links, either within the current document or to other documents. Note, that not all links in the HTML document can be placed inside the <nav> element; it can only include major navigation blocks.
  • <article> </article> The <article> element is used to define an independent, self-contained content (articles, blog posts, comments, etc.). The content of the element has its meaning, and it is easily differentiated from the rest of the webpage content.
  • <section> </section> The <section> element is used to group standalone sections within a webpage containing logically connected content (news block, contat information, etc.).

HTML Tag Collection

<img src=" " alt=" "> Is used to embed an image in a HTML page. The tag has two required attributes 'src' which specifies the path to the image (locally or URL) and 'alt', which specifies an alternate text in case the image cannot be displayed.
<a href=" "> </a> Defines a hyperlink, which is used to link from one page to another. Can also be used to jump to a specific part of the same page. The most important attribute is the 'href', which indicates the link's destination. The Hyperlinks clickable content is defined between the opening and closing tag.
